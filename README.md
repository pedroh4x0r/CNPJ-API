### Instalação Direta: :computer:<br/><br/>
**Clone o Repositório**
```bash
git clone https://github.com/pedroh4x0r/CNPJ-API
```

**Rode o composer** :fire:
```bash
composer install
```

**Agora é só executar**
```bash
Exemplo: php consultar.php
```

**OBS: mude o cnpj no source code**
<h1>CNPJ Default: 00.000.000/0001-91</h1>
